Feature Columns

Use feature columns to highlight key information of one or multiple sections. Add one of the following alignment classes to the feature column wrapper to change the alignment of icon and text: .left, .center, or .right.

Addtional Styles

Create feature boxes by combining boxes and feature columns. Alter box size by applying one of the following size classes to the box wrapper: .small, .medium, .large or .xlarge. Boxes can be styled further using a combination of background color/border-color, text color classes and border style classes. See Styling Classes for more information.
