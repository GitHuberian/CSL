Pricing Tables

Priced packages, options or services

Pricing tables are a great way of showing priced packages, options or serivices. Choose between two, three, four or five column pricing tables.

Addtional Styles

Style pricing tables by using a combination of size classes, background color and text color classes. See Styling Classes for more examples.

We can also use the .callout class for columns that we want to put forward. This class should be applied directly on the pricing table column wrapper as illustrated in the example below.

Note: Background color and text color classes are currently only supported for the table header, the table pricing section and table buttons. See Styling Classes for more information on combining color classes. See Buttons for more information on button styling. For more complex styling use CSS.
