Helper Classes

Useful Classes

A useful set of classes for altering content on smaller devices.

Floating Elements

Easily float elements by applying the .pull-left or .pull-right class to an element.

HTML
<div class="pull-right">Content...</div>
Clearing Floats

Clear floats by applying the .clearfix class to the wrapper of floated elements which will force it to exapand and push any floated elements below it. Alternatively, use the more old fashioned way for clearing floats by using the .clear class and applying to an element at the end of the floated elements.

.pull-left
.pull-left
.pull-left
HTML
<div class="box">
 <div class="clearfix">
  <div class="pull-left">Content...</div>
  <div class="pull-left">Content...</div>
 </div>
 <div class="pull-left">Content...</div>
</div>
Clearing Floats on Mobile New

Apply the .clear-float-on-mobile class to clear an elements float for screen widths ≤ 768px.

HTML
<div class="box pull-left clear-float-on-mobile">
 Content...
</div>
Remove Padding New

Remove the padding of a section block or element by applying the .no-padding-top, .no-padding-right, .no-padding-bottom, .no-padding-left or .no-padding class to an element. Use .no-padding-on-mobile to remove all padding for screen widths ≤ 768px.

The outer box will have its padding removed on mobile.
HTML
<div class="box pull-left clear-float-on-mobile">
 Content...
</div>
Text Alignment

Change text alignment of a section or element by applying the .left, .center, or .right class to an element. Use .left-on-mobile, .center-on-mobile or .right-on-mobile to change text alignment for screen widths ≤ 768px. This is useful when content flows from top to bottom on smaller screens.

This text will remain centered on desktop and mobile

This text will left align on mobile

HTML
<div class="center">Content...</div>
<div class="center left-on-mobile">Content...</div>
Vertical Alignment

Vertical alignment classes are useful for rollovers or captions in particular but can be used for any element that employs the use of "table" and "table-cell" for display. Timber rollovers and captions are by default set to vertical align middble and so we can use the .v-align-top or .v-align-bottom class to alter an elements vertical alignment. Use .v-align-top-on-mobile or .v-align-bottom-on-mobile to change vertical alignment for screen widths ≤ 768px. This is useful when images become smaller and the caption needs to be repositioned.

This captions is vertically aligned bottom on desktop but top on mobile.

HTML
<div class="thumbnail">
 <img src="images/thumbnail.jpg" width="825" height="400">
 <div class="caption-over-outer">
  <div class="caption-over-inner v-align-bottom v-align-top-mobile center color-white">
   <p>Caption</p>
  </div>
 </div>
</div>
Hiding/Showing

Force hide/show any element by applying the .hide or .show class to an element. Use .hide-on-mobile or .show-on-mobile to change an elements display for screen widths ≤ 768px.

This box shows on desktop but is hidden on mobile. Resize your browser!
This box is hidden on desktop but shows on mobile
HTML
<div class="hide-on-mobile">Content...</div>
<div class="hide show-on-mobile">Content...</div>
Margins

Remove an elements margins by appling the .no-margins class or .no-margin-top, .no-margin-right, .no-margin-bottom, .no-margin-left class to remove a specific margin of an element.

HTML
<div class="no-margins">Content...</div>
Remove Transition

Apply the class .no-transitions to remove an elements transition. Use .no-transition-on-mobile class to prevent an elements transition at screen widths ≤ 768px.

No transition No transition on mobile
HTML
<a href="#" class="button rounded no-transition">Button</a>
<a href="#" class="button rounded no-transition-on-mobile">Button</a>
