List Groups

Grouped lists are created by adding the class .list-group to the ul or ol. It can be styled further by adding the class .boxed which will add a border to all its children. Like other compoenents groups lists can be altered in size by applying on of the following size classes: .small, .medium, .large and .xlarge.
