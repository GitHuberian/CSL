The Grid

Columns

A 12 column responsive grid for easy layout creation

The grid is constructed of horizontal rows that define the max-width of the page, in this case 1140px. The rows form the outer layer of the layout, while the columns, create the inner, more complex layouts.

Nesting

You can easily nest columns to create complex layouts.

4
4
6
6
HTML
<div class="row">
 <div class="column width-4">...</div>
 <div class="column width-4">...</div>
 <div class="column width-4">
  <div class="row">
   <div class="column width-6">...</div>
   <div class="column width-6">...</div>
  </div>
</div>

Offset

Easily move sections up to eleven columns by using the .offset-xx class.

4
4
4
HTML
<div class="row">
 <div class="column width-4">...</div>
 <div class="column width-4 offset-4">...</div>
 <div class="column width-4 offset-8">...</div>
</div>

Push/Pull

Rearrange columns with ease using .push and .pull classes irregardless of the order in which they appear in the markup.

.push-2
.pull-10
HTML
<div class="row">
 <div class="column width-10 push-2">...</div>
 <div class="column width-2 pull-10">...</div>
</div>
