Getting Started

Timber is easy to use and offers 20 useful components for quick layout creation and 8 useful add-on plugins.

To get started simply include the necessary resource files and start creating. Timber requires jQuery v.1.11+ and timber.min.js.

CSS

<link rel="stylesheet" href="css/timber.css">

JS

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js"></script>
<script type="text/javascript" src="js/timber.min.js">

Add-on Plugins

A group of useful jQuery based plugins that will make your design come to life.
Use them in combination with Timber or on their own.

Avalance Slider

Avalanche is a 100% reponsive and CSS3 transitions driven slider with support for images, HTML5, YouTube and Vimeo videos

View

Summit Lightbox

A responsive lightbox with support for images, YouTube videos, Vimeo videos, inline HTML, iframes and ajax content.

View

Snowbridge Parallax

Snowbridge is a small plugin used for creating super smooth, fixed or fullscreen responsive parallax sections.

View

Horizon

Horizon is a small plugin that reveals elements as they enter the viewport using custom CSS3 transitions.

View

Equalize Heights

Quickly equalize heights of grouped or individual elements in the page. Grouped elements are evaluated on a per row basis so that each row stays nice and neat.

View

Retinize Images

Retinize is small plugin that checks whether a device is retina enabled, if so, it will swap out images in the page for retina sized images.

View

Counter

Counter is a small plugin that will count up or down to a given target number. Used in combination with horizon, counting can be forced to begin once it appears in the view port.

View

Swipe

Swipe is a plugin that enables the detection of left and right swipe gestures on touch devices. Used for Avalanche and Summit.

View
